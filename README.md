# 💳 Invoice Fraud Detection

## 🚀 Overview
This project detects fraudulent invoices using machine learning.  
It analyzes invoice patterns and flags suspicious transactions.

---

## 🧠 Model
- Random Forest Classifier
- SMOTE for handling imbalance

---

## 📊 Performance
- Accuracy: ~89%
- Balanced precision and recall

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

---

## ▶️ How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

---

## 💡 Features
- Detects fraud in real-time
- User-friendly interface
- Handles imbalanced data
